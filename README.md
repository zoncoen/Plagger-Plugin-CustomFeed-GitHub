# NAME

Plagger::Plugin::CustomFeed::GitHub - Custom feed for GitHub

# SYNOPSIS

    - module: CustomFeed::GitHubFeed
        config:
          token: {github_api_token}
          users:
            - {github_username}

# DESCRIPTION

Plagger::Plugin::CustomFeed::GitHub fetches public timeline for any user.
